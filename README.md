# 🔥 Pathfinding Visualizer (A* vs Dijkstra)

A multi-language pathfinding engine with visualization.

## 🚀 Features

- A* algorithm
- Dijkstra algorithm
- Interactive grid (Java GUI)
- Player movement (WASD)
- Maze building (mouse)
- Speed + scale controls

## 🧠 Algorithms

- Dijkstra → explores all directions evenly
- A* → uses heuristic (Manhattan distance)

## ▶️ Run

### Java (Visualizer)
./scripts/run_java.sh

### C++ (Algorithm)
./scripts/run_cpp.sh

### Python (Optional)
./scripts/run_python.sh

## 🎮 Controls

- WASD / Arrow Keys → move player
- Mouse → place/remove walls
- Button → switch A* / Dijkstra
